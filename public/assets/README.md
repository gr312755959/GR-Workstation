# 网站素材放这里

后续更新作品时，可以把图片、视频、文档都放在这个目录。

建议分类：

- `videos/混剪视频/`：混剪、情绪片、节奏向作品
- `videos/宣传片/`：企业宣传片、项目宣传片、课程宣传片
- `videos/AI视频/`：AI短剧、AI生成视频、AI辅助分镜作品
- `videos/短视频/`：口播、科普、信息流、账号内容
- `videos/活动拍摄/`：会议、活动、直播、现场记录
- `videos/hero-background.mp4`：以后如果想把首页底图换成磨砂感视频，可以先放这个文件名
- `covers/`：作品封面图
- `photos/`：个人工作照、现场照、设备照
- `docs/`：简历、项目说明、PDF案例

命名建议：

- `2026-项目名称-封面.jpg`
- `2026-项目名称-成片.mp4`
- `2026-项目名称-说明.pdf`

目前第一版使用：

- `hero-abstract.png`：当前首页背景图备份
- `hero-editorial.png`：新版首页背景图备份
- `hero-ai-studio.png`：旧版首页背景图备份

技术说明：

- 网站实际调用的首页背景图在 `src/assets/hero-abstract.png`
- 这样打包后双击 `dist/index.html` 也更容易正常显示
- 首页代码已经预留视频背景位置，后续需要时可以把视频接进去，并保留现在的磨砂遮罩层
